gnss_lib_py
===========

.. _mainpage:

This is a helpful explanation that explains the purpose of this
repository.

This repository allows you to load receiver data from:

    * `Google Android Derived Dataset <https://www.kaggle.com/c/google-smartphone-decimeter-challenge>`__
    * rinex
    * nmea

And satellite ephemeris data from:

    * sp3


This may also include some cool images of successful algorithms.
